Angular Pipes List

Playground

  • DecimalPipe: Formats numbers with a fixed number of decimal places => {{ number | number : "1.2-2" }}
  • PercentPipe: Formats numbers as percentages => {{ number | percent : "1.2-2" }}
  • KeyValuePipe: Iterates over object properties and provides key-value pairs => Key: {{ number.key }}, Value: {{ number.value }}
  • JsonPipe: Converts an object to a JSON string for debugging purposes => {{ numbers | json }}
  • Custom Pipe => Sum of list: {{ numbers | reduce : sum : 0 }}
  • AsyncPipe: Handles observable or promise data => Promise: {{ p | async }}
  • DatePipe: Formats dates. You can use it to display dates in various formats => Clock: {{ time$ | async | date : "hh:mm:ss" }}
  • Clock: {{ time$ | async | date : "medium" }}
  • CurrencyPipe: Formats numbers as currency values => {{ euro | currency : "USD" : "symbol" : "1.2-2" }}
